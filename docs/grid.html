Fellas Grid

Grid layouts

  • grid-layout-nosidebar: Creates a vertical grid with a navbar on top, a main area for content, and a footer in the bottom.
  • grid-layout-sidebar-left: Creates a grid with a navbar, a sidebar, a main area, and a footer. Alternatively, use grid-layout-sidebar-right for moving the sidebar to the right.
  • grid-nav: refers to the navbar
  • grid-main: refers to the main content area
  • grid-footer: refers to the footer area
  • grid-sidebar: refers to the sidebar regardless of position right or left.
